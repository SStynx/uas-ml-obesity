# Sumber Dataset Publik

Dokumen ini mencatat asal-usul, legalitas, serta sitasi dataset publik yang digunakan dalam pengerjaan proyek Tugas Akhir/UAS Machine Learning ini.

## Detail Informasi Dataset
* **Nama Resmi Dataset:** *Estimation of obesity levels based on eating habits and physical condition*
* **Platform Penyedia:** UCI Machine Learning Repository
* **Tautan Unduh Resmi:** [https://archive.ics.uci.edu/dataset/544/estimation+of+obesity+levels+based+on+eating+habits+and+physical+condition]
* **Karakteristik Data:** Multivariate (Memiliki banyak fitur campuran nominal dan numerik).
* **Jumlah Baris Data:** 2.111 records.
* **Jumlah Fitur/Kolom:** 17 atribut (16 fitur prediktor + 1 variabel target).

## Karakteristik Pembuatan Data
Dataset ini merupakan gabungan data riwayat klinis nyata (sebesar 23% yang diperoleh secara langsung dari kuesioner pengguna di wilayah negara Meksiko, Peru, dan Kolombia) dengan data sintetis tambahan (sebesar 77% yang digenerasikan secara artifisial menggunakan algoritma SMOTE melalui *Weka Tool*). Penambahan data sintetis ini dilakukan oleh peneliti asli untuk menangani permasalahan ketidakseimbangan kelas (*imbalanced class*) sebelum dipublikasikan ke publik.
